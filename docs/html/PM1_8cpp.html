Embeds text into an image using the Plus-Minus One (PM1) method.

This function takes an image path, text to embed, and an output image name. It converts the text to binary and embeds it into the blue channel of the image pixels using the PM1 method. The embedding process can be described by the formula:

\[ P_i' = \begin{cases} P_i \pm 1, & \text{if } P_i \mod 2 \neq m_i \\ P_i, & \text{otherwise} \end{cases} \]

where ( P_i ) is the pixel value before embedding, ( P_i' ) is the pixel value after embedding, and ( m_i ) is the bit of the message.

Parameters
imagePathPath to the input image file.
textText to embed into the image.
imageNameName of the output image file with embedded text.
Returns
int Returns 0 on success, -1 if the image cannot be loaded.
