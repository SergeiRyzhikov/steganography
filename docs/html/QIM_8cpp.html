Embeds text into an image using the Quantization Index Modulation (QIM) method.

This function takes an image path, text to embed, and an output image name. It converts the text to binary and embeds it into the blue channel of the image pixels using quantization with a specified quantization step.

Формула встраивания:

\[ P_i' = q \cdot \left\lfloor \frac{P_i}{q} \right\rfloor + \frac{q}{2} \cdot m_i, \]

Формула извлечения:

\[ m_i = \arg \min_{e \in \{0,1\}} \left| P_i' - P_{i,e}'' \right|, \]

Parameters
imagePathPath to the input image file.
textText to embed into the image.
imageNameName of the output image file with embedded text.
Returns
int Returns 0 on success, -1 if the image cannot be loaded.
