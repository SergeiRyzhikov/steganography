\doxysubsection{Detailed Description}
Main entry point for the steganography program. 

This file contains the main function which allows the user to choose between different steganography methods (PM1, QIM, LSB) and perform embedding or extraction of text in/from images.

Main function to drive the steganography program. 

This function allows the user to select a steganography method and either embed text into an image or extract text from an image using the chosen method.

The following methods are supported\+:
\begin{DoxyItemize}
\item {\bfseries{LSB (Least Significant Bit)}}\+: \[ P_i = p_7p_6p_5p_4p_3p_2p_1p_0, \quad P_i\textnormal{\textquotesingle} = p_7p_6p_5p_4p_3p_2p_1m_i \]
\item {\bfseries{PM1 (Plus-\/\+Minus One)}}\+: \[ P_i\textnormal{\textquotesingle} = \begin{cases} P_i \pm 1 \cdot r, & \text{if } P_i \mod 2 \neq m_i; \\ P_i, & \text{otherwise}, \end{cases} \]
\item {\bfseries{QIM (Quantization Index Modulation)}}\+: \[ P_i\textnormal{\textquotesingle} = q \cdot \left\lfloor \frac{P_i}{q} \right\rfloor + \frac{q}{2} \cdot m_i \]
\end{DoxyItemize}
