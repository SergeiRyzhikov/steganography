{\footnotesize\ttfamily int QIM\+\_\+embed (\begin{DoxyParamCaption}\item[{string}]{image\+Path,  }\item[{string}]{text,  }\item[{string}]{image\+Name }\end{DoxyParamCaption})}



Embeds text into an image using the Quantization Index Modulation (QIM) method. 

This function takes an image path, text to embed, and an output image name. It converts the text to binary and embeds it into the blue channel of the image pixels using quantization with a specified quantization step.

Формула встраивания\+: \[ P_i\textnormal{\textquotesingle} = q \cdot \left\lfloor \frac{P_i}{q} \right\rfloor + \frac{q}{2} \cdot m_i, \]

Формула извлечения\+: \[ m_i = \arg \min_{e \in \{0,1\}} \left| P_i\textnormal{\textquotesingle} - P_{i,e}\textnormal{\textquotesingle}\textnormal{\textquotesingle} \right|, \]
